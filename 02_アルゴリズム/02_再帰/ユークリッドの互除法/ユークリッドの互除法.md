# ユークリッドの互除法（euclidean algorithm）

## アルゴリズム

2つの整数$m, n$の最大公約数を求める。

1. $m$を$n$で割った余りを$r$とする
1. $r=0$であれば、最大公約数は$n$となり、処理を終了とする
1. $r \neq 0$の場合は、$m \leftarrow n, n \leftarrow r$として、1. に戻る

## 実装

### C++

```cpp
#include<iostream>
using namespace std;

// 最大公約数（greatest common divisor）
int GCD(int m, int n) {
    if (n == 0) return m;
    
    return GCD(n, m % n);
}

int main() {
    cout << GCD(51, 15) << endl;
    cout << GCD(15, 51) << endl;
}
```
