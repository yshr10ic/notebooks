# 線形代数

## スカラーとベクトル

スカラー（scaler）とは１つの数値からなる変数のこと
$x$のように小文字で表す

### 数式

$$
x = 10
$$

### Python

```python
x = 10
```

### R

```R
x <- 10
```

ベクトル（vector）とは大きさに加えて方向を表す複数のスカラーからなる変数のこと
${\rm x}$のように太字の小文字で表す

### 数式

$$
{\rm x} = \binom{x_1}{x_2} = \binom{1}{2}
$$

### Python

```python
x = np.array([1.0, 2.0])   # 1行2列
y = np.array([3.0], [4.0]) # 2行1列
```

### R

```R
x <- c(1.0, 2.0)
y <- matrix(c(3.0, 4.0), nrow=2, ncol=1)
```
